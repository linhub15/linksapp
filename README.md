# Linktree Clone

An over architected clone of linktree to showcase systems design and bleeding
edge technology.

Uses [Deno 2](https://deno.com/).

### Buzzwords

- Hexagonal, Clean Architecture, Screaming Architecture
- RESTful API
- DB ORM
- Monorepo
- Integration Tests

### Backend Tech

- Hono
- Zod
- Drizzle ORM
- OpenApi
- Scalar API Docs
- Turso LibSQL (sqlite)

### Frontend Tech

- Vite + React
- Tailwind v4
- Tanstack Query
